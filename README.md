# Eventlyy — Homepage prototype

Open `index.html` in a browser to view the responsive homepage prototype.

Features included:
- Responsive layout: mobile-first with a fixed bottom nav and desktop top nav.
- Dark / light theme toggle (persists in `localStorage`).
- Featured carousel with scroll-snap and JS controls.

To run locally, simply open the file in your browser or use a simple static server:

```
# Python 3
python -m http.server 8000

# Node (if you have serve)
serve .
```
